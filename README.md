#MVHS Währungsrechner#

Das Android-Projekt wird für den Kurs "Einblick in die Programmierentwicklung für Android Smartphones" an der Münchner Volkshochschule verwendet. Der Quellcode und die Dokumentation sind frei verfügbar. Weiterentwicklung ist erwünscht.
